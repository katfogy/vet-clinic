### Prerequisites

you have to those tools in your local machine.

### Setup

Clone the project.

```sh
  cd my-folder
  git clone git@github.com:katfogy/vet-clinic.git
```
Go to the project directory.
```sh
 cd vet-clinic
```
In the project directory, run/ Excute each query in the following order:
```sh
  Excute schema.sql
  Excute data.sql
  Excute queries.sql
```
